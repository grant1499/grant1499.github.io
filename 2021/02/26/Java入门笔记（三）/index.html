4.Java中的方法

记住：Java中的方法都是值传递。其实 Java 只存在一种传参的方式，就是用实参给形参赋值。

5.方法的重载（非常重要）

方法重载基本类似C++中的函数重载。

重载的特殊之处在于：如果参数满足自动类型转换的方法有好几个，重载规则是选择最近的调用。

比如：int add(int a)和int add(double b)，如果传入byte,short,int类型的参数a，会调用第一个，传入float,double类型的参数a会调用第二个。自己写的话最好避免这样的情况。
