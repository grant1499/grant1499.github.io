Java异常机制（Exception）

错误（Error）与异常（Exception）

Exception可以分成运行时异常和非运行时异常。

以上理论只作为了解内容，重点是如何处理异常。

异常处理（抛出与捕获）

try-catch…-catch结构：一般把较大的异常放在后面。

假设要捕获多个异常，必须从小到大，否则会报错！
